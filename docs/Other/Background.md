# Background
This project originally started in 2022

## Why make Dream
Dream wasn't originally a programming language; it was just a differential calculator written in Java.

Back then, I was obsessed with Minecraft servers and wanted to create an unbeatable NPC. 
At first, I just tried to make it work purely through logic, but no matter how I coded it, 
the NPC always acted really weird.

Later, I went online to learn a lot of things, and I understood the deeper concepts of AI. I tried to implement this NPC using AI methods.

So at first I wrote a weight calculator in Java, but it was calculating everything wrong and was super slow.

So I started making constant changes. During this process, I kept searching for all kinds of stuff on the browser. Later, this calculator could barely calculate correctly, but it was also then that I realized if I wanted to really improve this calculator, I would be facing an insurmountable mountain. So from that time on, I put this project aside (2022).

## From 2024 to 2026
At this point, I realized that AI seemed to be on the rise, because many well-known AIs had emerged, like ChatGPT.

I restarted this project and rewrote it in C first. At that time, I learned about PyTorch, and I realized that I probably couldn't surpass it. PyTorch is ahead of Dream both in ecosystem and technology.

To mimic PyTorch, I went to study compiler principles. I know this might be the wrong choice.

After turning it into a language, I kept improving it, and along the way, I kept searching online for all kinds of problems and knowledge.

It has continued like this until now. Later, I named it Dream. Dream currently has about 30,000 to 35,000 lines of code.

Open sourcing it was also because, in the end, I felt there was no point in continuing with Dream; I really couldn't change anything.
